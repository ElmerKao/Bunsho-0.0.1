Bunsho

Inicio

Bunsho es un editor de texto plano hecho como proyecto de aprendizaje, Bunsho esta basado en un editor llamado Kilo hecho por Salvatore Sanfilippo siguiendo la guía que se encuentra en internet.

Esta guía se puede encontrar enlazada en el GitHub the Bunsho, aunque algunas funciones han sido añadidas aparte.

Características

Bunsho esta programado en C, la parte del usuario es mayoritariamente en inglés para mayor alcanze de usuarios, es ligero, de fácil uso y amigable para el usuario, aunque algunas características siguen en desarrollo, estas a lo largo del tiempo se irán añadiendo y poniendo a disposición en esta página.

Descarga

1- Descomprime el archivo .zip

2- Acceda a la carpeta descomprimida

3- Luego instale bunsho con el gestor de paquetes dpkg

sudo dpkg -i bunsho_0.1_all.deb

4- Listo! Para usar Bunsho escribe en la terminal
